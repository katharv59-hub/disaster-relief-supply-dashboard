# Disaster Relief Supply Dashboard

This project is a Disaster Relief Supply Monitoring System built using Flask, Pandas, and NumPy.

## Project Overview

The system monitors supply levels across multiple cities and identifies critical shortages.  
It calculates the number of days supplies will last based on stock and consumption rate.

## Features

- Tracks supply levels across cities
- Calculates days remaining for supplies
- Generates alerts for critical cities
- Visualizes supply trends using charts
- Monthly allocation tracking
- City-wise stock monitoring

## Technologies Used

- Python
- Flask
- Pandas
- NumPy
- HTML
- CSS
- JavaScript
- Chart.js

## How to Run

1. Clone the repository

git clone https://github.com/katharv59-hub/disaster-relief-supply-dashboard.git
